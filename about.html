About Me

My name is Naina, and I am a final-year Bachelor of Computer Applications (BCA) student with a deep passion for technology and software development. Throughout my academic journey, I've worked on a variety of projects that have allowed me to hone my skills in programming, web development, and database management.

One of my key projects is a Portfolio Website, a clean and responsive site built with HTML, CSS, and JavaScript. It showcases my professional resume, highlighting my skills, experience, and achievements through a user-friendly interface.

Another project I'm proud of is a Movie Search Website, a dynamic web application that integrates with a movie API. This app features real-time search functionality, detailed movie information pages, and a 'Favorite' system with persistent storage, all within a responsive design that works seamlessly across devices.

Additionally, I developed an Invoice Generator as a command-line application in C. This project includes user input for customer and product details, along with automatic calculations of subtotals and total amounts, showcasing my ability to work with core programming concepts and logic.

Skills

  • Programming Languages: Python, Java, C
  • Web Development: HTML, CSS, JavaScript
  • Databases: MySQL
  • Operating Systems: Windows, Linux

I am continuously seeking opportunities to apply my knowledge in real-world scenarios and am excited to contribute to innovative projects in the tech industry. Please feel free to explore my projects and reach out if you'd like to collaborate!
